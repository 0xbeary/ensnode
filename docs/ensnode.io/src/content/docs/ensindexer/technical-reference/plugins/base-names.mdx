The Basenames plugin indexes Basenames (.base.eth).

### Data sources

The Basenames plugin has a single data source, `Basenames`.

### Indexed contracts

The Basenames plugin indexes events and data from the following smart contracts relevant to the Basenames deployment:

- Registry
- BaseRegistrar
- EARegistrarController
- RegistrarController
- Resolver
