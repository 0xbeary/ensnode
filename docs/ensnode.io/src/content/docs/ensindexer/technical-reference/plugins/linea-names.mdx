The Lineanames plugin indexes Linea Names (.linea.eth).

### Data sources

The Lineanames plugin has a single data source, `Lineanames`.

### Indexed contracts

The Lineanames plugin indexes events and data from the following smart contracts relevant to the Lineanames deployment:

- Registry
- BaseRegistrar
- EthRegistrarController
- NameWrapper
- Resolver
